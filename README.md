# Minesweeper AI

A Minesweeper game implementation with AI training capabilities.

## Setup

1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`

## Run
```bash
python3 main.py
```

## Project Structure

- `game/` - Minesweeper game logic and UI
- `ai/` - AI agent implementation
- `main.py` - Entry point
